
LiDAR Traffic Intelligence Platform

An interactive web application for analyzing Ouster LiDAR traffic data.

## Features

- Interactive trajectory visualization
- Real-time trajectory playback
- Object filtering
- Traffic query engine
- Turning Movement Counts (TMC)
- Time-to-Collision (TTC)
- Post-Encroachment Time (PET)
- Safety analytics
- AI Research Assistant
- CSV export
- Interactive dashboards

## Technologies

- HTML
- CSS
- JavaScript
- Chart.js
- Leaflet
- Ouster LiDAR
- Computer Vision
- Traffic Safety Analytics

## Applications

- Intelligent Transportation Systems
- Smart Cities
- Connected Infrastructure
- Traffic Operations
- Road Safety Analysis
